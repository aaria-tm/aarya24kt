
# Aarya24kt Shopping Website

A modern e-commerce frontend website where users can browse products and manage a shopping cart with real-time updates.

---

## 🛍️ Features

- Browse products across categories:
  - Men
  - Women
  - Special Offers
- Add products to cart
- Increase / decrease item quantity
- Automatic total price calculation
- View all selected items in cart
- Clean and user-friendly interface

---

## 🛠️ Tech Stack

- HTML  
- CSS  
- JavaScript  

---

## ⚙️ How It Works

- Users can select products from different sections  
- Clicking "Add to Cart" adds items instantly  
- Quantity buttons update item count dynamically  
- Total price updates automatically based on cart changes  

---

## 🚀 How to Run Locally

1. Download or clone the repository  
2. Open `index.html` in your browser
